# MemosYou

**MemosYou** —— 你的个人灵感记录工具。

一款为 Android 打造的 [✍️memos](https://github.com/usememos/memos) 客户端，支持自托管服务器同步，也可纯本地离线使用。基于上游 Moe Memos 进行 Material 3 Expressive 风格改造。

## 安装

从 [Releases](https://github.com/NecoArc-Chaos/MemosYou/releases) 下载 APK 安装。

## 特色

- 📝 随时随地记录灵感，像给自己发推文一样
- ☁️ 支持连接自托管 ✍️memos 服务器同步，也可纯本地使用
- 🏷️ 标签，置顶，轻松整理
- 😀 表情反应 & 评论（探索页）
- 🔒 离线优先，恢复网络后自动同步
- 🖼️ Markdown 编辑与渲染，图片附件，待办事项
- 🔐 无数据收集，隐私优先

分支特色:
- 🎨 Material 3 Expressive 设计：动态取色、多套预设主题、波浪进度条、骨架屏加载
- 贴表情/评论系统
- 更多服务器端元素设计

## 开发

基于 Kotlin + Jetpack Compose 开发。
使用 Deepseek V4 Pro 进行 Vibe Coding
